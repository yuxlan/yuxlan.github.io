关于页面布局(flex、盒模型、浮动)

2017-06-10

作业部落链接地址

css盒模型

Flex布局

<!-- 任何一个容器都可以指定为flex布局 -->
.box{
    display: flex;
}

<!-- 行内元素也可以使用flex布局 -->
.box{
    display: inline-flex;
}

<!-- Webkit内核的浏览器必须加上-webkit前缀 -->
.box{
    display: -webkit-flex;
    display: flex;
}

<!-- 设置flex布局后，子元素的float/clear/vertical-align属性将全部失效 -->
  • 容器属性

1.flex-direction决定主轴的方向

.box{
    flex-direction: row(主轴水平，起点为左) | row-reverse(主轴水平，起点为右) | column(主轴垂直，起点为上) | column-reverse（主轴垂直，起点为下);
}

2.flex-wrap决定当项目无法在主轴上一行排列时，如何换行

.box{
    flex-wrap: nowrap(默认不换行) | wrap(换行第一行在上面) | wrap-reverse(换行第一行在下面);
}

3.flex-flow是flex-direction + flex-wrap，默认值为row nowrap

4.justify-content定义了项目在主轴上的排列方式

.box{
    justify-content: flex-start(左对齐) | flex-end(右对齐) | center(居中) | space-between(两端对齐项目之间间隔相等) | space-around(每个项目两侧之间的距离相等);
}

5.align-items定义项目在交叉轴上的对齐方式

.box{
    align-items: flex-start(起点对齐) | flex-end(终点对齐) | center(居中对齐) | baseline(项目第一行文字对齐) | stretch(默认占满整个容器的高度);
}

6.align-content定义多根轴线对齐方式，如果只有一根轴线就不起作用

.box{
    align-content: flex-start(起点对齐) | flex-end(终点对齐) | center(居中) | space-between(两端对齐所有轴线之间间隔相等) | space-around(所有轴线之间间隔相等) | stretch(默认占满整个交叉轴);
}
  • 项目属性

1.order定义项目排列顺序，从小到大，默认为0

.item{
    order: 整数数值;
}

2.flex-grow定义项目的放大比例，默认为0

.item{
    flex-grow: 数值;
}

3.flex-shrink定义项目的缩小比例，默认为1

.item{
    flex-shrink: 数值;
}

4.flex-basis定义项目原定占据的主轴空间，默认为auto，即项目原大小

.item{
    flex-basis: 
}

5.flex是flex-grow + flex-shrink + flex-basis，默认为0,1,auto，后两者可选

6.align-self允许单个项目和其它项目不一样的对齐方式，可覆盖align-item，默认为auto，即继承容器的align-item

.iten{
    align-self: auto | flex-start | flex-end | center | baseline | stretch;
}
