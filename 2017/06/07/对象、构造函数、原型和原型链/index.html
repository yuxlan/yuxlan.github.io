对象、构造函数、原型和原型链

2017-06-07

作业部落链接地址

  • 工厂模式
    在需要创建多个对象时，写一个模子，以此复制对象，减少重复创建对象的过程。
var createPerson = function(name,age){
// 声明一个中间对象，即工厂模式的模子
var o = new Object();
// 依次添加属性和方法
o.name = name;
o.age = age;
o.getName = function(){
    return this.name;
}
return o;
}

// 创建实例
var perJack = createPerson("Jack",21);
var perTom = createPerson("Tom",24);

麻烦：

  1. 没法识别对象实例的类型
  2. 不停地为getName方法分配空间
  • new

关于new做了什么：

  1. 声明一个中间对象；
  2. 将该中间对象的原型指向构造函数的原型；
  3. 将构造对象的this指向该中间对象；
  4. 返回该中间对象，即返回实例对象；
// 创建函数
var Person = function(name,age){
    this.name = name;
    this.age = age;
    this.getName = function(){
        return this.name;
    }
}

// 将构造函数以参数形式传入
function New(func){
    // 声明一个中间对象，该对象为最终返回的实例
    var res = {};
    if(func.prototype !== null){
        // 将实例的原型指向构造函数的原型
        res._proto_ = func.prototype;
    }

// ret为构造函数执行的结果，通过apply，将构造函数内部的this指向修改为指向res，即为实例对象
var ret = func.apply(res,Array.prototype.slice.call(arguments,1));

// 在构造函数中明确指定了返回对象时，new的执行结果就是返回该对象
if((typeof ret === "object" || typeof ret === "function") && ret !== null){
    return ret;
}

// 如果没有明确指定返回对象，则默认返回res,res就是实例对象
return res;
}

// 通过new声明创建实例，p1实际接收的是new中返回的res
var p1 = New(Person,"Tom",20);
console.log(p1.getName());

// 也可以判断出实例的类型
console.log(p1 instanceof Person);// true（instanceof识别实例类型）

  • 原型
// 声明构造函数
function Person(name,age){
    this.name = name;
    this.age = age;
}

// 通过prototype属性，将方法挂载到原型对象上
Person.prototype.getName = function(){
    return this.name;
}

var p1 = new Person("Tom",10);
var p2 = new Person("Tim",20);

console.log(p1.getName === p2.getName); // true



当我们访问实例对象中的属性和方法时，会优先访问实例对象本身的属性和方法。

function Person(name,age){
    this.name = name;
    this.age = age;
    this.getName = function(){
        console.log("this is");
    }
}

Person.prototype.getName = function(){
    return this.name;
}

var p1 = new Person("Tom",20);
p1.getName(); // this is



通过in来判断，一个对象是否拥有某个属性/方法，无论是该属性/方法存在于实例对象还是原型对象。

function Person(name,age){
    this.name = name;
    this.age = age;
}

Person.prototype.getName = function(){
    return this.name;
}

var p1 = new Person("Tom",20);
console.log("name" in p1); // true
