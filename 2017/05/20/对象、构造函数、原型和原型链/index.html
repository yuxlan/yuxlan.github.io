  1. 没法识别对象实例的类型
  2. 不停地为getName方法分配空间
  • new

关于new做了什么：

  1. 声明一个中间对象；
  2. 将该中间对象的原型指向构造函数的原型；
  3. 将构造对象的this指向该中间对象；
  4. 返回该中间对象，即返回实例对象；
